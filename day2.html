Operators & Expressions

Operators perform operations on values to form expressions:

  • Arithmetic: +, -, *, /, %, ++, --
  • Assignment: =, +=, -=, *=, /=
  • Comparison: ==, ===, !=, !==, <, >, <=, >=
  • Logical: && (AND), || (OR), ! (NOT)
// Example: let x = 5; let y = 10; let sum = x + y; // 15 let isEqual = (x == y); // false

Conditional Statements

if statement

if (condition) {
        // code if condition true
      } else {
        // code if condition false
      }

switch statement

switch(expression) {
        case value1:
          // code
          break;
        case value2:
          // code
          break;
        default:
          // code
      }

Loops in JavaScript

for loop

for (let i = 0; i < 5; i++) {
        console.log(i);
      }

while loop

let i = 0;
      while (i < 5) {
        console.log(i);
        i++;
      }

do-while loop

let i = 0;
      do {
        console.log(i);
        i++;
      } while (i < 5);

Task: Find Factorial of a Number

The factorial of a number n (n!) is the product of all positive integers up to n.

0! = 1 by definition.

function factorial(n) {
  let result = 1;
  for (let i = 1; i <= n; i++) {
    result *= i;
  }
  return result;
}

// Example usage:
console.log("5! =", factorial(5)); // Output: 120
