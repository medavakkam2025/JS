Introduction to JavaScript

JavaScript (JS) is a powerful, dynamic programming language primarily used to create interactive effects within web browsers. It allows you to add functionality to your web pages.

  • Client-side scripting language
  • Used for manipulating HTML and CSS
  • Supports event-driven, functional, and imperative programming styles
  • Runs on all modern web browsers without any setup

JavaScript Placement

1. Inline JavaScript

JavaScript code can be written directly inside an HTML tag using the onclick or other event attributes.

// Example: Inline JavaScript <button onclick="alert('Hello World!')">Click Me</button>

2. Internal JavaScript

Within a <script> tag inside the <head> or <body> sections of an HTML document.

<script> alert('Hello World!'); </script>

3. External JavaScript

JavaScript code in a separate .js file linked using the <script src="..."> tag.

<script src="myscript.js"></script>

Advantages of external JS:

  • Code reuse across multiple pages
  • Cleaner HTML files
  • Easier to maintain

Variables in JavaScript

Used to store data values. JavaScript has three main ways to declare variables:

Declaration Scope Re-assignable? Use Case
var Function or globally scoped Yes Legacy code, avoid in modern projects
let Block scoped Yes Preferred for variables that change value
const Block scoped No (constant reference) Use for variables that should not change

Example of variable declarations:

let name = 'Alice';
const PI = 3.14159;
var isOnline = true;

Data Types in JavaScript

JavaScript supports several data types classified as follows:

  • Primitive Types:
    • String, Number, BigInt
    • Boolean
    • Undefined
    • Null
    • Symbol
  • Non-Primitive Types: Object, Array, Function

Type Conversion

JavaScript automatically or manually converts between data types:

Conversion Example Result
String to Number Number('123') 123 (number)
Number to String (123).toString() '123' (string)
Boolean to Number Number(true) 1
Implicit Conversion '5' + 10 '510' (string concatenation)

Task: Calculate Simple Interest

Simple Interest formula: SI = (P × R × T) / 100

Create a JavaScript script that calculates simple interest given:

  • P - Principal amount (number)
  • R - Rate of interest per annum (%)
  • T - Time in years

Example code:

function calculateSimpleInterest(P, R, T) { let SI = (P * R * T) / 100; return SI; } // Example usage: let principal = 5000; let rate = 5; let time = 3; console.log("Simple Interest is:", calculateSimpleInterest(principal, rate, time));

Interview Questions & Answers

Question Answer
What are the different ways to declare a variable in JavaScript? Using var, let, and const.
What is the difference between var and let? var is function-scoped and allows hoisting; let is block-scoped and not hoisted in the same way.
Can you reassign a const variable? No, const variables cannot be reassigned, but objects assigned to const can have their properties changed.
What are primitive data types in JavaScript? String, Number, BigInt, Boolean, Undefined, Null, Symbol.
Explain type coercion in JavaScript. Automatic or implicit conversion of values from one data type to another, e.g., when adding a string and a number.
Where can you place JavaScript code in an HTML document? Inline within HTML tags, inside internal <script> tags, or as an external file linked with <script src>.
What happens if you declare a variable without var, let, or const? It becomes a global variable (in non-strict mode), which is generally a bad practice.
How do you convert a string to a number in JavaScript? Using Number(string) or the unary plus operator +string.
